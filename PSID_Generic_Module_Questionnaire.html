PSID Generic Crisis Module - Expanded Foundation Questionnaire

This standalone demo is intentionally longer than the strict scored core. It preserves the 7 always-on ranked items, adds broader employment, housing, aid, and wellbeing prompts, and presents a generic front-end module that feels complete even before any crisis-specific toggle is added.

Scored core7 questions
Expanded demo18 questions
Estimated duration12-14 min
Intended useAny crisis type

Design note: The analytic model still deploys 7 Generic Core items in the selected 28-question module. This HTML intentionally expands the generic front-end to 18 questions so stakeholder demos show a richer, more interview-ready baseline instrument irrespective of crisis type.
